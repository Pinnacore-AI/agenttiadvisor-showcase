# AgenttiAdvisor™ Showcase

Tämä sovellus on Agenttiarmeija™ Builderin (Suomen Manus/Lovable) ensimmäinen virallinen demonstraatio.

## Projektin Tiedot
- **Nimi**: AgenttiAdvisor™
- **Teknologia**: HTML5, Vanilla CSS3 (Custom Design System), JavaScript
- **Teema**: Glassmorphism / Futuristic Dark Mode
- **Rakentaja**: Agenttiarmeija™ Swarm (Architect, Frontend, Intel Agents)

## Ominaisuudet
1. **Live Intelligence**: Reaaliaikaiset markkinadatapäivitykset (SOL/USD).
2. **Swarm Status**: Näyttää aktiivisten agenttien tilan ja toiminnan.
3. **Advanced Aesthetics**: Dynaaminen tausta, lasiefektit ja sulavat animaatiot.

## Ohjeet
Sovellus on optimoitu livenä katseltavaksi.
